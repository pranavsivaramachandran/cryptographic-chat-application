# Cryptographic Chat Application

A simple encrypted chat application written in Python.

## Features

- End-to-end symmetric encryption using Fernet (AES-based)
- TCP socket communication
- Local network demonstration
- Educational project

## Installation
pip install -r requirements.txt


Generate a key:
generate_key.py

Start the server:
server.py

Start the client:
client.py

## Disclaimer

This project is intended for educational purposes and is **not** a production-ready secure messaging system.
